وصف المشروع (Project Description)
​هذا المشروع عبارة عن نظام محادثة ذكي (Chatbot) مصمم للإجابة على التساؤلات بناءً على قاعدة بيانات مخصصة. يعتمد المشروع على تقنيات معالجة اللغات الطبيعية (NLP) باستخدام نموذج AraBERT لضمان فهم دقيق للغة العربية، مع واجهة تفاعلية عبر منصة Telegram.
​🏗️ هيكل المشروع (Project Structure)
​إليكِ شرح لمكونات الملفات التي تظهر في المشروع:
​smart_chatbot.py: المحرك الأساسي للبوت الذي يحتوي على منطق المعالجة والذكاء الاصطناعي.
​telegram_bot.py: ملف الربط مع واجهة التلجرام لإرسال واستقبال الرسائل من المستخدمين.
​arabert_model.py: الكود المسؤول عن تحميل واستخدام نموذج AraBERT لمعالجة النصوص العربية.
​qa_data.json: قاعدة البيانات التي تحتوي على الأسئلة والأجوبة (Dataset).
​generate_10k_dataset.py: سكربت مخصص لتوسيع قاعدة البيانات وتوليد بيانات تدريب ضخمة.
​requirements.txt: قائمة المكتبات البرمجية اللازمة لتشغيل المشروع (مثل transformers, torch, python-telegram-bot).
​updata_bot.py: ملف مسؤول عن تحديث بيانات البوت أو تطوير وظائفه.
​final_generator.py: الجزء المسؤول عن توليد الإجابات النهائية بشكل منطقي.
​🚀 المميزات (Key Features)
​دعم اللغة العربية: استخدام نماذج لغوية متقدمة مدربة خصيصاً للعربية.
​التفاعل الفوري: إمكانية الوصول للبوت من أي مكان عبر التلجرام.
​قاعدة بيانات مرنة: سهولة تحديث الأسئلة والأجوبة عبر ملف JSON.
​معالجة ذكية: القدرة على فهم سياق السؤال وليس فقط الكلمات المفتاحية.
